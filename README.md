# Smart Song Collector & Playlist Manager (Python)

## Overview
This project is a Python-based song collection and playlist management system.
It allows users to organize, filter, and manage music data efficiently using structured logic and automated sorting.

## Features
- Add and remove local songs
- Sort songs by different parameters
- Search and filter functionality
- Add Cover playlist for each song
- Playlist management
- Structured data handling

## Technologies Used
- Python
- Object-Oriented Programming (if you used it)
- File handling / data structures
